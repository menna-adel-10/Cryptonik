# Cryptonik 🚀

A modern cryptocurrency trading dashboard built with Angular, designed to display real-time market data and provide users with an interactive and responsive trading interface.

## ✨ Features
- Real-time cryptocurrency data integration using REST APIs
- Interactive UI for monitoring price trends and market movements
- Responsive design optimized for desktop and mobile devices
- Modular and scalable component-based architecture
- Efficient data handling and state management

## 🛠 Tech Stack
- Angular (v16)
- TypeScript
- HTML5 / CSS3 / Sass
- REST APIs

## 🚀 Key Highlights
- Built reusable and scalable UI components for dynamic data rendering
- Optimized performance for handling frequent API updates
- Designed clean and maintainable frontend architecture
